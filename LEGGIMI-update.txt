-----------------------------------------------------------------
PASSO 1 — Applicare i file aggiornati
-----------------------------------------------------------------
Dalla RADICE del tuo checkout locale (la cartella che contiene "tools/" e
"launcher.py"):

    unzip -o multimedia-master-update.zip

L'opzione -o sovrascrive senza chiedere. I file nuovi vengono creati.

-----------------------------------------------------------------
PASSO 2 (opzionale) — Rimuovere le cache obsolete
-----------------------------------------------------------------
Nel main erano committate delle cache compilate e un log che la versione nuova
ha rimosso. NON servono e Python le rigenera da solo. Puoi cancellarle senza
rischi. Dalla radice del progetto:

    find tools -name "__pycache__" -type d -exec rm -rf {} +
    rm -f tools/audio_manager/crash.log

(Se non le rimuovi non succede nulla di grave: sono solo file .pyc rigenerati.)
